# Webhard

基于 web 技术的硬件调试工具集合

地址：https://webhard.pages.dev

## 功能

1. Serial
2. Camera
3. Microphone
4. Bluetooth
5. ST-Link
6. DAPLink
7. STM32USB-DFU
8. HID

## 开发

- 安装依赖

```sh
npm install
```

- 运行（编译&热更新）

```sh
npm run dev
```

- 编译（打包）

```sh
npm run build
```

- 格式化

```sh
npm run lint
```
