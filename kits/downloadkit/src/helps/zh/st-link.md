# st-link 模式（st-link / ST-Link）

此模式使用 ST-Link 调试器通过 SWD 接口写入目标芯片 Flash（一般基于 WebUSB）。

## 准备条件

- 浏览器支持 WebUSB（建议 Chrome / Edge / Arc，且需要安全上下文）。
- ST-Link 的系统驱动/固件状态正常（通常无需额外安装驱动，但某些平台可能需要）。
- 正确连接 SWD：
  - `SWDIO`（常见为 `PA13`）
  - `SWCLK`（常见为 `PA14`）
  - `GND` 必须共地

## 连接方式

- 将 ST-Link 连接到目标板的 SWD 接口。
- 建议同时接上 `NRST`（如果你的目标板电路/烧录流程需要硬复位）。
- USB 线连接 ST-Link 到电脑（用于 WebUSB/设备权限申请）。

## 网页操作步骤

1. 在主页选择模式为 `st-link`。
2. （可选）根据目标板/信号质量调整 `SWCLK` 速率（页面默认提供下拉项）。
3. 选择固件文件（支持 `*.hex` / `*.bin`）。
4. 点击「下载固件」。
5. 浏览器会弹出 WebUSB 设备授权请求，选择你的 ST-Link 设备并确认。
6. 等待进度条完成；成功后通常会触发复位（重启目标程序）。

## 常见问题

- **设备授权窗口找不到 ST-Link**
  - 确认 ST-Link 已正确插入并供电
  - 重新插拔 USB
  - 尝试更换 USB 口/线材

- **写入失败**
  - 降低 `SWCLK` 速率再试
  - 检查 SWDIO/SWCLK 是否接反或虚焊
  - 确认目标芯片处于可被调试/擦写的状态（如需要清除保护/解锁）

## 浏览器要点

- 需要安全上下文：HTTPS 或 localhost。
- WebUSB 在 Chromium 系列浏览器更可靠。
